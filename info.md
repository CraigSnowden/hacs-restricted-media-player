# Restricted Media Player

Wrap existing media player entities and restrict the visible sources to a user-selected subset.

## Features

- Filter media player sources to show only allowed options
- Show "Technician Mode" when base player is on hidden source
- Full pass-through of all media controls
- Easy UI-based configuration
- Update sources anytime through options flow

Perfect for hiding technical inputs (HDMI, AUX) while keeping streaming apps visible.
